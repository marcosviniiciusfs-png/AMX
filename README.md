# AMX Simulador

Simulador de crédito AMX criado em Vite, React, TypeScript, shadcn-ui e Tailwind CSS.

## Desenvolvimento

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Variaveis de ambiente

O envio de leads usa uma URL de webhook configurada fora do codigo:

```env
VITE_FORM_WEBHOOK_PROXY_URL=
```

Este repositório nao inclui configuracao Wrangler, Cloudflare Worker, dominio CNAME ou integracoes herdadas de outros projetos.
